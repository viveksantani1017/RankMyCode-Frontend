## Tech Stack

NextJs, RTK Query, Tailwind CSS


## Run Locally

Clone the project

```bash
  git clone https://github.com/viveksantani1017RankMyCode-Frontend
```

Go to the project directory

```bash
  cd RankMyCode-Frontend-main
```

Install dependencies

```bash
  npm install
```

Start the NextJs Server

```bash
  npm run dev

```
